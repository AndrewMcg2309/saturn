Virtual reality therapy in paediatrics

Final year project and dissertation

  • Project: Virtual Reality Therapy in Paediatrics (VRT)
  • Technologies: Github, C#, Unity, Oculus VR
  • Link: https://github.com/AndrewMcg2309/VR-Anxiety-Therapy

Overview
Virtual Reality Therapy in Paediatrics (or VRT) was a virtual realty application to help anxious children
receiving intramuscular injections. The focus of VRT was for the deltoid muscle of the arm which is the most common for vaccines. VRT targeted children above the age of 8 years old to assist them in the process of getting injections.

The main objective of VRT is to provide a comfortable virtual environment that will help children who
get anxious or nervous while receiving injections to distract them from what is happening and
immerse them in a virtual world.

VRT was essentially a game that would immerse the children enough that they wouldn't see nor feel the needle, while also making the game immersible enough without too much movement which would disrupt the procedure.

The dissertation was 23k words and 100 pages long. The VR application and dissertation received a first as a grade.

Technologies and Methodologies

Feature-Driven Development was the best course of action in terms of software
development. This meant each feature would be added one at a time. Using Git to create
branches allowing for these features to be implemented and if the feature failed, the
application could roll back to before the attempt was made.

Unity was used for the application as it was the most widely used and a powerful 3D game development engine. Unity also combines its built-in user interface with C# coding. Visual Studio was the default editor used.

Oculus Rift was the VR headset used and can be easily implemented with Unity to provide a virtual reality experience in your own application.

Full dissertation can be provided on request.
